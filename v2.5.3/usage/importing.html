                    Importing

                    Since transax supports both CommonJS and ES6 modules, you can import it using the package name. The correct path will be resolved automatically.

                    If you are using TypeScriptopen in new window, you can even import directly from the source files.

                    ESM
                    import { Generator } from 'transax';
import { Generator } from 'transax/esm'; // or this to be explicit
                    CommonJS
                    const { Generator } = require( 'transax' );
const { Generator } = require( 'transax/cjs' ); // or this to be explicit
                    TypeScript
                    import { Generator } from 'transax';
import Generator from 'transax/src/Generator/Generator'; // or this to be explicit

                    Warning

                    From this point on the documentation will assume you are using the ES6 Module syntax. If you are using CommonJS, you can still use the same API, but you will need to use the require syntax instead of import.
