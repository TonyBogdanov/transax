                                                                                                  Class: TernaryExpressionToken

                                                                                                  index.TernaryExpressionToken

                                                                                                  Represents a ternary expression that returns the result of the truthy expression if the test expression returns a truthy value, or the result of the falsy expression otherwise.

                                                                                                  Examples: a ? b : c( 0 < 1 ) ? b : c
