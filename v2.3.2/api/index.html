        # AnalyzerOptions

        Æ¬ AnalyzerOptions: Object

        Options for the Analyzer.

        # Type declaration

        NameTypeDescription
        keyFormatter?KeyFormatterA function to format the translation key based on the context. Defaults to ( key => key ).
        logger?LoggerInterfaceOptional logger instance. Defaults to new Logger( { namespace: 'TRANSAX:ANALYZER' } ).
        names?string[]The names of the functions to analyze. Defaults to [ '$t' ].

        # Catalog

        Æ¬ Catalog: Record<Locale, Dictionary>

        Catalog of translation entries, keyed by locale.

        Example

        {
  "en_US": {
    "my.translation.key": "Hello there, nice to meet you!",
  },
}

        # Context

        Æ¬ Context: Object

        Runtime context to be used during translation.

        # Type declaration

        NameTypeDescription
        globals?ContextGlobalsHashmap of globals available to translation expressions.
        locale?LocaleThe locale to translate to. If not specified, the default locale specified in the translator options will be used.
        params?ContextParamsHashmap of parameters available to translation expressions.

        # ContextGlobals

        Æ¬ ContextGlobals: Record<string, any>

        # ContextParams

        Æ¬ ContextParams: Record<string, any>

        # Dictionary

        Æ¬ Dictionary: Record<Key, Value>

        Translation dictionary as a hashmap of key-value pairs.

        Example

        { "my.translation.key": "Hello there, nice to meet you!" }

        # GeneratorOptions

        Æ¬ GeneratorOptions: Object

        Options for the Generator.

        # Type declaration

        NameTypeDescription
        analyzer?AnalyzerInterfaceOptional analyzer instance. Defaults to new Analyzer().
        compiler?CompilerInterfaceOptional compiler instance. Defaults to new Compiler().
        logger?LoggerInterfaceOptional logger instance. Defaults to new Logger( { namespace: 'TRANSAX:GENERATOR' } ).
        translations?CatalogOptional translation catalog. Defaults to {}.
